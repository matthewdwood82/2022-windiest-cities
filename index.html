2022-windiest-cities

Per {nflfastR}, during the regular season last year which home NFL outdoor stadium(s) had the fastest wind speed recorded at kickoff?

library(tidyverse)
library(nflfastR)
library(DT)

df_schedule <- nflfastR::fast_scraper_schedules(2022) %>% 
  dplyr::select(game_id:gametime, away_team, home_team, location, roof, wind) %>% 
  dplyr::filter(game_type == "REG", location == "Home", roof == "outdoors") 

df_answer <- df_schedule %>% 
  dplyr::arrange(desc(wind)) %>%
  dplyr::mutate(rank = dplyr::row_number()) %>% 
  dplyr::select(rank, gameday, week, gametime, away_team, home_team, wind)

DT::datatable(df_answer, rownames = FALSE)

NOTE: {nflfastR} does have some missing wind data for the 2022 season, so above should not be considered exhaustive. Most of the missing data is from early in the season however. Wind tends to pick up late in the fall/winter, so hopefully the missing data here does not include too many of the extreme values for which we are interested.
